# 2D-Platformer

Технологии: Unity 2D, корутины, Animator, Physics2D.

Архитектура: конечные автоматы (State) для игрока и врагов, фабрики состояний, интерфейсы IDamageable, IReplenishable.

Системы: здоровье с различными индикаторами (плавные, текстовые), AI врагов (патруль, преследование, атака), сбор предметов (монеты, аптечки), навык вампиризма с UI-индикатором.

Паттерны: State, Factory, Observer, Strategy, Object Pool (респавн монет).
